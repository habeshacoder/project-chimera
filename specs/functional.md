## User Stories

- As a Network Operator, I can set goals (e.g., "Promote fashion in Ethiopia") via dashboard, so Planner decomposes into tasks.
- As an Agent (Planner), I monitor trends via MCP Resources, to trigger content creation.
- As a Worker, I generate multi-modal content (text/image/video) with consistency lock, using MCP Tools.
- As a Judge, I validate outputs with confidence >0.90 auto-approve, else escalate to HITL.
- As an Agent, I execute transactions via Coinbase AgentKit, checked by CFO Judge.
- As the System, I disclose AI nature in posts and responses.

## HITL Flow

- Escalated tasks queued in dashboard for approve/reject/edit.
